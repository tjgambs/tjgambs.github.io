CALCULATOR AID

JUNE 2, 2015
  • For my final project in AP Computer Science I built a calculator check in/out web application for the Maine Township High School South Math Department. The web application was programmed with Google Apps Script, JavaScript, HTML, and CSS. The purpose of the application was to help Math teachers check in/out calculators loaned to their students. The teachers where also able to use a barcode scanning app on there iDevice called ScanForURL which would make the check in/out process much easier.
