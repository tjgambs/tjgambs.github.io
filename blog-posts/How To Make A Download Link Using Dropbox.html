How to Make a Download Link Using Dropbox

January 19, 2015
  • 1. Create a dropbox account, Dropbox
    2. Log-in then on the left hand side of the screen right-click "Files"
    3. Click "Upload" and then proceed to upload the file of folders you want to have downloadable
    4. After the file is uploaded right-click on the file or folder and hover over "Share"
    5. Click "Send Link..."
    6. Copy the web address and change one thing
    7. Change "?dl=0" at the end of the web address to "?dl=1"

    Example

    My original link: https://www.dropbox.com/sh/kvl7te8zg75j08y/AACEEmnijbs2mRqizkZxNQSGya?dl=0
    My corrected link: https://www.dropbox.com/sh/kvl7te8zg75j08y/AACEEmnijbs2mRqizkZxNQSGya?dl=1

    8. Enter the corrected link into your search bar and when you press enter that file will begin to download.
