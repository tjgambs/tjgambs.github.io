How To Get Free Mac OS X NTFS Support

January 19, 2015
  • We will be using a program called "NTFS for Mac OS X" to achieve this. Let us begin!

    1. Download this
    2. Once you have it downloaded, navigate to your downloads folder.
    3. Open up the folder called "Paragon NTFS"
    4. Once open double click the file "ntfsmac_trial_u.dmg" and install it.
    5. DON'T REBOOT OR UNMOUNT INSTALLER DMG
    6. Instead, open Terminal window and drop "install_ntfs12_refresh.sh" there.
    7. Hit Return. Say yes then type in your admin password.
    8. Reboot and now you have NTFS support on Mac OS X forever.
    9. I noticed if you delete the "Paragon NTFS" folder it will no longer work, so hang onto that folder and you should be fine!

    This is a trial version of the software however the trial will reset and you will never need to buy the software. Enjoy native NTFS support for Mac!
